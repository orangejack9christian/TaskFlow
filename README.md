## Quick Start

### Option 1: Use Locally
Just open `index.html` in your browser!

### Option 2: Access from Other Devices (Local Network)

**Windows:**
1. Double-click `start-server.bat`
2. Share the IP address shown with devices on your Wi-Fi
3. Access from any device: `http://[YOUR-IP]:8000`

**Requirements:** Python 3.x (download from python.org if needed)

### Option 3: Deploy Online (Access from Anywhere)

See `DEPLOY.md` for detailed instructions on:
- GitHub Pages (free, easy)
- Netlify (drag & drop)
- Vercel (one command)

## Keyboard Shortcuts

- `Ctrl+K` / `Cmd+K` - Add new item
- `Ctrl+E` / `Cmd+E` - Export data
- `Ctrl+I` / `Cmd+I` - Import data
- `?` - Show all shortcuts
- `Esc` - Close modals

## Data Storage

TaskFlow stores all data locally in your browser. Your data stays private and secure on your device.
